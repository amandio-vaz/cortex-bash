### Executar localmente

**Pré-requisitos:**  Node.js

- 1.Instale as dependências:
   `npm install`
- 2.Defina o`GEMINI_API_KEY`em [.env.local](.env.local) para sua chave API Gemini
- 3.Execute o aplicativo:
   `npm run dev`
